# task-manager-python_project
A simple python task manger with productivity tracking
## Features
- Add tasks
- View tasks
- Mark as completed
- Delete tasks
- Tasks saved in JSON file
## Project Structure
- 'main.py'-Runs the program
- 'task-manager.py'- task logic(add, delete ,complete)
- 'storage.py'-handles saving/ loading tasks
## How to Run
python main.py
